Parsing Methods

The tool tries parsers in this order and uses the first that succeeds:

1
Column-Aware PDF Parser
Splits multi-column PDFs by x-position (Devereaux, Granite Valley, etc.)
2
Table + Species-Header Parser
Handles pdfplumber tables with species embedded in headers (Tigerton, etc.)
3
Line-by-Line Text Parser
Reads plain-text PDFs line by line, tracking species section headers (Brenneman, etc.)
4
OCR Fallback
Converts scanned/image PDFs to text via Tesseract OCR (CC Cook, etc.)
AI
AI Parser (GPT-4o)
Manually triggered per-file. Requires OpenAI API key above.
